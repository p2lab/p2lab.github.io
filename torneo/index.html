Reglas

  • Para apuntarte, comunícalo a través del canal #competitivo de Discord. Además, debes decir el equipo con el que combatirás (solo los nombres de los Pokémon).
  • Puedes hacer tantos cambios como quieras (en tu equipo) antes de que comience el torneo. Una vez empiece, el equipo debe permanecer inalterado.
  • Este torneo es para las ediciones Pokémon Espada y Escudo, por lo que necesitarás una suscripción a Nintendo Switch Online para participar.
  • El formato de juego será singles OU.
    • No se puede tener más de un Pokémon con el mismo número de Pokédex Nacional.
    • No se puede dormir a más de un Pokémon al mismo tiempo.
    • Están prohibidos los movimientos OHKO (Fisura, Guillotina, Perforador, Frío Polar).
    • No se puede usar un Pokémon con las habilidades Veleta o Sombra Trampa.
    • No se puede usar Doble Equipo, Reducción ni Relevo.
    • No se pueden usar Pokémon Uber (pulsa aquí para ver la lista de Ubers).
    • No se puede usar Dinamax/Gigamax.
  • El resultado de un enfrentamiento se decidirá al mejor de tres.
  • Los dos usuarios que se enfrenten deben confirmar el resultado, o bien aportar un vídeo usando la función de grabación de Switch donde se vea el resultado.
  • Esta página es meramente informativa, todo se gestionará desde el canal #competitivo del servidor de MikaCookies. Para apuntarte o comunicar los resultados, por favor, hazlo allí.
